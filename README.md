## Setup

1. Clone the repository
2. Update the content in `index.html`:
   - Add your real project URLs
   - Update contact information (email, social links)
   - Add your live instance URLs
3. Customize colors in `styles.css` (CSS variables at the top)
4. Deploy to:
   - GitHub Pages (free)
   - Vercel (free)
   - Netlify (free)
   - Render (free)

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... more colors */
}
```

### Projects
Add/remove project cards in the `#projects` section of `index.html`

### Instances
Update live instance cards in the `#instances` section
